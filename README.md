# [truelockmc Video Downloader](https://github.com/truelockmc/video-downloader) CLI-only Builds

This Repositories only purpose is to offer compiled cli-only releases.
Every UI Component has been removed, and the releases are compiled without ```--windowed```.

The main goal is to have smaller Release Files and offer stdin/out support for windows.

### [CLI Usage](https://github.com/truelockmc/video-downloader/blob/main/Docs/CLI.md)
